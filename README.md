## 🚨 常见避坑类型

- 销售承诺不兑现（如学位/交房标准）
- 周边环境与宣传严重不符
- 公摊过大/户型设计不合理
- 隐性捆绑（车位、装修包）
- 维权频发或开发商口碑差
